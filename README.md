Welcome to my dotfiles repository!

## Acknowledgements

## Dependencies

* `curl`
* `git`
* root access
* build-essential for Linux and Developer Tools for macOS

## Installation

sh -c "$(curl -fsSL https://raw.githubusercontent.com/alericcardi/dotfiles/main/.config/yadm/install.sh)" "" --decrypt
